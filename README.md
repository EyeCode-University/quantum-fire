<h1>📝 Тема: Компоненты, состояние, поиск, работа с данными, интерактивный интерфейс</h1>
Example:Название задания: Secret Healing Recipes

Описание задачи
Необходимо создать веб-приложение на React + Next.js, в котором будут храниться секретные рецепты лечения популярных болезней.

Приложение должно представлять собой интерактивную базу знаний, где пользователь может искать нужную болезнь и видеть рецепт лечения.

Главная цель задания — научиться:

• работать с компонентами
• организовывать структуру данных
• реализовывать поиск
• делать интерактивный интерфейс

Функционал приложения

В приложении должна быть база рецептов, например:

Простуда

Головная боль

Боль в горле

Кашель

Бессонница

Усталость

Стресс

Каждый рецепт должен содержать:

Название болезни
Описание симптомов
Рецепт лечения
Список ингредиентов
Рекомендации

Поиск

На странице должен быть поисковый input.

Пользователь вводит название болезни и приложение мгновенно фильтрует список рецептов.

Поиск должен работать интерактивно (live search) без перезагрузки страницы.
Обязательные требования

Использовать Next.js App Router

Создать компоненты:

RecipeCard
SearchBar
RecipeList
RecipePage

Рецепты должны храниться в массиве объектов.
