## Суть задания

Создай React-приложение с **работающей пагинацией**. Тебе нужно отобразить большой список данных (минимум 50 элементов) и разбить его на страницы. Пользователь должен иметь возможность переключаться между страницами.

---

| Тема | Что отображаем |
|------|----------------|
| 🎬 **Фильмы** | Каталог фильмов с рейтингами и постерами |
| 📚 **Книги** | Библиотека книг с авторами и обложками |
| 🎵 **Музыка** | Плейлист треков с исполнителями |
| 🏨 **Отели** | Список отелей с рейтингами и ценами |

## 📋 Пошаговый план

### 1️⃣ Подготовь данные

Создай MockAPI на https://mockapi.io с **минимум 50 записями**.

**Пример для фильмов:**
```json
{
  "id": "1",
  "title": "Inception",
  "director": "Christopher Nolan",
  "year": 2010,
  "rating": 8.8,
  "poster": "https://example.com/inception.jpg"
}
```

**Пример для книг:**
```json
{
  "id": "1",
  "title": "1984",
  "author": "George Orwell",
  "year": 1949,
  "pages": 328,
  "cover": "https://example.com/1984.jpg"
}
```

> ⚠️ Минимум **50 записей** для полноценной пагинации

---

### 2️⃣ Реализуй пагинацию

| Элемент | Описание |
|---------|----------|
| **Количество на странице** | 10 элементов на одной странице |
| **Кнопки навигации** | "Предыдущая", "Следующая", номера страниц |
| **Активная страница** | Визуально выделена текущая страница |
| **Отключенные кнопки** | "Предыдущая" на первой странице, "Следующая" на последней |
| **Счётчик** | Показывай "Страница X из Y" или "Элементы 1-10 из 50" |

---

### 3️⃣ Логика пагинации

```javascript
// Основные переменные
const [currentPage, setCurrentPage] = useState(1);
const itemsPerPage = 10;

// Расчёт данных для текущей страницы
const indexOfLastItem = currentPage * itemsPerPage;
const indexOfFirstItem = indexOfLastItem - itemsPerPage;
const currentItems = data.slice(indexOfFirstItem, indexOfLastItem);

// Общее количество страниц
const totalPages = Math.ceil(data.length / itemsPerPage);
```

---

### 4️⃣ Интерфейс пагинации

Твоя пагинация должна выглядеть примерно так:

```
┌─────────────────────────────────────────┐
│  [←] [1] [2] [3] ... [10] [→]          │
│        Страница 2 из 10                 │
└─────────────────────────────────────────┘
```

Или вариант с диапазоном:

```
┌─────────────────────────────────────────┐
│  [Предыдущая]  1 2 3 4 5  [Следующая]  │
│        Показано 11-20 из 50             │
└─────────────────────────────────────────┘
```
